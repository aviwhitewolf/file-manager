# @uncia/file-manager

A modern, accessible file manager UI component library built with React, Radix UI, and Tailwind CSS.

## Packages

- [`@uncia/file-manager`](./packages/ui) - The main UI component library

## Quick Start

```bash
# Install the library
pnpm add @uncia/file-manager

# Import styles in your app
import '@uncia/file-manager/styles.css';

# Use components
import { Button, AlertDialog } from '@uncia/file-manager';
```

## Development

```bash
# Install dependencies
pnpm install

# Build the library
cd packages/ui && pnpm build

# Run the demo app
pnpm dev
```
